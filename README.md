# Code for Matsubara - Official Website

🌐 **Official Site:** [https://codeformatsubara.org](https://codeformatsubara.org)

大阪府松原市のオープンデータを活用し、市民の力で地域をアップデートする非営利のシビックテック・プロジェクト「Code for Matsubara」の公式ウェブサイト用リポジトリです。

## 概要 (Overview)
「テクノロジーとデザインの力で、地域をもっと便利で楽しくする」をコンセプトに活動しています。
市役所が公開するオープンデータを利用し、本当に使いやすいシステムやアプリを自らの手（コード）で開発・無償提供することで、地域のDX（デジタルトランスフォーメーション）を草の根から推進します。

## プロジェクト (Projects)
当コミュニティでは、以下のオープンソース・プロジェクトを開発・公開しています。

* 📍 **松原市 温故知新AIマップ (`matsubara-onko-chishin-map`)**
  * 松原市公式のオープンデータ「まつばらいろはかるた」を活用した、子ども向け歴史探索Webアプリ。GPSとAIを活用し、地域の文化財を分かりやすく解説します。（週末ハッカソンにて実装）
* 🔍 **松原市 AI議事録検索 (`matsubara-council-ai-search`)** *※In Development*
  * 松原市議会の膨大な議事録をRAG（検索拡張生成）で解析し、要点を可視化するAI検索エンジン。

## 技術スタック (Tech Stack)
* HTML5 / CSS3
* Tailwind CSS (CDN)
* JSON-LD (Semantic Data)
* Hosted on GitHub Pages
